#Bullet Read
Bullet Read è una webapp minimal per leggere testi in modalità RSVP (Rapid Serial Visual Presentation): invece di scorrere con gli occhi su righe e paragrafi, mostra una parola alla volta in un punto fisso dello schermo, con una riga di contesto sotto che evidenzia la parola corrente.

Funzionalità
Lettura parola-per-parola con velocità regolabile (WPM).

Play / Pausa.

Avanti / Indietro di una parola.

Evidenziazione della “lettera focus” (stile ORP) nella parola grande.

Riga di contesto sotto con evidenziazione della parola corrente.

Controlli da tastiera:

Spazio: play/pausa

Freccia sinistra/destra: parola precedente/successiva
